Global desktop files:
/usr/share/applications/

Local desktop files:
~/.local/share/applications/

Example of a desktop file:
[Desktop Entry]
Name=MyApp
Exec=/usr/bin/vivaldi-stable --ozone-platform=wayland --enable-features=UseOzonePlatform --use-cmd-decoder=validating %U
Type=Application

Reload the desktop database:
update-desktop-database ~/.local/share/applications
