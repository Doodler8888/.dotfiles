Delete the last commit without deleting it's changes:
git reset --soft HEAD~1

Check commit history:
git log (there is also 'git reflog')

Rename branch:
git branch -m <new_branch_name>

Cancel uncommited changes in the current directory (modifications, delections,
new files):
git checkout -- .

Remove remote branch (write branch without 'origin/'):
git push origin --delete <branch_name>

Push after commit reset/deletion:
git push --force-with-lease
