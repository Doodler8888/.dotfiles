Launch bare nvim:
nvim --clean

Launch neovim with a custom file (it's not a clean launch with a specific file, try to add '--no-plugin'):
nvim -u <file_path>

Show higlight group name under the cursor:
:Inspect

Show value of a color group:
:highlight <hg_name>
Example:
:highlight @punctuation.delimiter

Move between curly brackets:
[/]+m

Reload buffer:
e! - "!" forces the reload, discarding any unsaved changes.

Search in specific file patterns:
:vimgrep /oldword/g file1.txt file2.txt src/**/*.lua

Interactively replace found pattern in each file (you don't have to use vimgrep
before, you just need to fill quickfix list in any way, like using C-q in
telescope):
:cfdo %s/oldword/newword/gc | update

Update plugins (to confirm update run ':write' in the output window):
lua vim.pack.update()

Update a plugin (to confirm update run ':write' in the output window):
lua vim.pack.update({"<plugin_name>.nvim"})
