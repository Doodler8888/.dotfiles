Connect to a database:
psql -h pg-primary.internal.company.com -U your_username -d name_of_db

Connection url:
postgresql://myuser:mypassword@localhost/mydatabase

Print current user:
SELECT user;

List all roles:
\du

List all databases:
\l or \list (without semicolon)

Create backup (example):
sudo docker exec --user postgres sonarqube-db-1 pg_dump -p 25432 -d \
sonarqube_db2 -F c -b -v -f <backup_path>

Restore backup:
cat path/to/dump | docker compose exec -T postgres pg_restore -U sentry -d \
sentry --clean --if-exists
