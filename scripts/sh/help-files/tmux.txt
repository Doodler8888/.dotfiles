Source config:
tmux-source /path/to/config/file

Attach session:
tmux attach-session -t <session_name>

Cmd line:
<prefix> + :

Create a new tab:
<prefix> + c

Kill a tab:
<prefix> + &

Create a vertical split:
<prefix> + %

Create a horizontal split:
<prefix> + "

Cycle between splits:
<prefix> + o

Rename tab:
<prefix> ,
