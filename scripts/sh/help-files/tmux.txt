Source config:
tmux-source /path/to/config/file

Cmd line:
<prefix> + :

Create a new tab:
<prefix> + c

Kill a tab:
<prefix> + &

Create a vertical split:
<prefix> + %

Create a horizontal split:
<prefix> + "

Kill a split:
<prefix> + x

Cycle between splits:
<prefix> + o

Rename tab:
<prefix> ,

Copy mode:
<prefix> + [
